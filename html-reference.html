HTML Element

Description

All HTML code goes within the HTML open and end tag

Syntax

    <html>
      <head>
        Abstract content
      </head>
      
      <body>
        Visual content
      </body>
    </html>

Head Element

Description

Contains the invisible aspect of your website such as the title element of the page, and referencing a stylesheet.

Syntax

      </head>

Description

Syntax

Syntax Output

Heading Elements

Description

Creates a bolded line of text on a new line. Can choose from h1 through h6 for different sizes. h1 is the largest and h6 is the smallest

Syntax

<h1>Heading Content</h1>

Example

Syntax Output

<h1>This is a heading</h1>

Table Element

Description

Creates a table with rows and columns

Syntax

        <table>
          <tr>
            <th>C1R1</th>
            <th>C2R1</th>
          </tr>
          <tr>
            <td>C1R2</td>
            <td>C2R2</td>
          </tr>

Example

C1R1 C2R1
C1R2 C2R2

W3Schools Extra Information

Block Element

Description

Groups together a block of code for the purpose of applying a property over all of the block of code at once.

Syntax

        <div>
          Block of code
        </div>

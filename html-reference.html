HTML Reference Page

Paragraph Element

Description

Creates a paragraph on a new line.

Syntax

<p>Paragraph Content</p>

Example

Syntax Output

<p>This is a paragraph</p>

This is a paragraph

Heading Element

Description

Creates a bolded line of text on a new line. Can choose from h1 through h6 for different sizes. h1 is the largest and h6 is the smallest

Syntax

<h1>Heading Content</h1>

Example

Syntax Output

<h1>This is a heading</h1>

This is a heading

Title Element

Description

Creates a description of the page at the top of the tab

Syntax

<title>Title Content</title>

Example

Syntax Output

<title>This is a title</title>

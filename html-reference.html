HTML Reference Page

HTML Section

HTML Element

Description

All HTML code goes within the HTML open and end tag

Syntax

    <html>
      <head>
        Abstract content
      </head>
      
      <body>
        Visual content
      </body>
    </html>

Head Element

Description

Contains the invisible aspect of your website such as the title element of the page, and referencing a stylesheet.

Syntax

      <head>
        Abstract Content
      </head>

Title Element

Description

Creates a description of the page at the top of the tab

Syntax

<title>Title Content</title>

Example

Syntax Output

<title>This is a title</title>

Body Element

Description

The body element usually contains all of the content on your website

Syntax

<body>
Website Content
</body>

Example

Syntax Output

<body>
<h1>Heading</h1>
<p>Paragraph</p></title>
</body>

Paragraph

Heading Elements

Description

Creates a bolded line of text on a new line. Can choose from h1 through h6 for different sizes. h1 is the largest and h6 is the smallest

Syntax

<h1>Heading Content</h1>

Example

Syntax Output

<h1>This is a heading</h1>

This is a heading

Paragraph Element

Description

Creates a paragraph on a new line.

Syntax

<p>Paragraph Content</p>

Example

Syntax Output

<p>This is a paragraph</p>

This is a paragraph

Break Element

Description

Syntax
